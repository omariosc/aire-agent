# Intel oneAPI
<!-- Brief introduction -->
Intel oneAPI is a comprehensive, open, and standards-based programming model designed to simplify development across diverse computing architectures. [Official Documentation](link_to_official_documentation)

## The Intel module on Aire

| Version |  Command                    |
|---------|-----------------------------|
| X.Y.Z   | `module load intel/X.Y.Z`   |
| A.B.C   | `module load intel/A.B.C`   |

:::{note}
Previous versions of Intel oneAPI included both `ifort` and `ifx` compilers. However, starting from the 2025 version, only `ifx` is supported.
:::
