# VTK

VTK is an open-source software system for processing and visualising scientific data. For further information, usage instructions and tutorials, please see the [official documentation](https://docs.vtk.org/en/latest/).

## The VTK module on Aire

Aire currently offers two versions of VTK, one with and one without OpenMPI support:

| Version           |  Command                                                      |
|-------------------|---------------------------------------------------------------|
| 9.3.1             | `module load vtk/9.3.1/gcc-14.2.0_hdf5-1.14.5`                |
| 9.3.1 (OpenMPI)   | `module load vtk/9.3.1/gcc-14.2.0_openmpi-5.0.5_hdf5-1.14.5`  |
